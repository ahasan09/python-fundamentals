# Python Fundamental

Collection of Python learning scripts covering core Python fundamentals with no external dependencies.

## Tech Stack
- Python 3 (standard library only)

## Project Structure
```
python_fundamental/
└── *.py    # Individual scripts per topic
```

## Development
```bash
# Run any script directly
python <script>.py
```
No installation or virtual environment required.

## Key Notes
- Pure standard library — no pip installs needed.
- Covers topics like data types, control flow, functions, OOP, and file I/O.
