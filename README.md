# Python Fundamentals

A collection of standalone Python scripts covering core language fundamentals. Each file is a self-contained lesson with examples you can run directly.

## Topics Covered

| File | Topic |
|------|-------|
| `variables.py` | Variable declaration and assignment |
| `numbers.py` | Integers, floats, arithmetic operators |
| `string.py` | String methods, slicing, formatting |
| `escape-sequence.py` | Escape characters (`\n`, `\t`, etc.) |
| `comparison-operator.py` | `==`, `!=`, `<`, `>`, `<=`, `>=` |
| `logical-operator.py` | `and`, `or`, `not` |
| `conditional-statement.py` | `if`, `elif`, `else` |
| `ternary-operator.py` | Inline conditional expressions |
| `for-loop.py` | `for` loop and `range()` |
| `while-loop.py` | `while` loop |
| `function.py` | Defining and calling functions |
| `type-convertion.py` | `int()`, `str()`, `float()`, `bool()` |
| `calculator.py` | Simple CLI calculator |
| `app.py` | Combined demo |

## Prerequisites

- [Python](https://www.python.org/) 3.x

## Running the Scripts

```bash
# Run any file directly
python3 variables.py
python3 string.py
python3 calculator.py
```

No external libraries required — all scripts use the Python standard library only.
